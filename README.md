Rubik's 2x2 – Кубик Рубика 2×2 на 7 языках
Мини-кубик Рубика в терминале!
Вращай грани, собирай комбинации, соревнуйся с таймером.
Реализован на 7 языках программирования с единым интерфейсом, цветным выводом и поддержкой всех стандартных ходов.

🚀 Возможности
6 граней – U, D, L, R, F, B (верх, низ, лево, право, перед, зад).

Повороты – по часовой стрелке и против (U, U', D, D', …).

Перемешивание – случайная последовательность ходов.

Отображение – цветная 3D-развёртка кубика.

Проверка сборки – определяет, собран ли кубик.

Таймер – засекает время с начала сессии.

Интерактивный режим – ввод команд в реальном времени.

Пакетный режим – передача последовательности ходов как аргумент.

Рекорды – сохраняет лучшее время (опционально).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [ходы] [опции]
Аргументы
ходы – строка с последовательностью ходов (например, "R U R' U'").

Если ходы не указаны, запускается интерактивный режим.

Команды (в интерактивном режиме)
U, U', D, D', L, L', R, R', F, F', B, B' – повороты граней.

scramble – перемешать кубик.

reset – сбросить в собранное состояние.

display – показать текущее состояние.

solve – проверить, собран ли кубик.

quit – выход.

Примеры
bash
# Интерактивный режим
python rubik.py

# Выполнить последовательность ходов
python rubik.py "R U R' U'"

# Перемешать и показать
python rubik.py scramble

🧠 Архитектура
Кубик представлен шестью гранями (Front, Back, Up, Down, Left, Right).
Каждая грань – это матрица 2×2 цветов (0–5).
Повороты граней реализованы через перестановку элементов внутри грани и обновление соседних граней.
Все реализации используют одинаковую логику и цветовую схему.

✨ Дополнительные фичи
Цветной вывод – цвета граней отображаются ANSI-кодами.

ASCII-развёртка – компактное отображение всех шести граней.

Проверка сборки – мгновенная обратная связь.

Случайное перемешивание – 20 ходов по умолчанию.
